# ADHD向け 1分アイドリング式タイマー

## Testing / 手動テスト
1. 日付跨ぎで安全側リセット
   - 23:59にWORK開始 → 00:00を跨ぐ → 00:00以降に統計が当日0から始まる
   - 00:00直後の`startBreak`で前日分が加算されない
1. LocalStorage破損フォールバック
   - LocalStorageの該当キーを文字列`"invalid"`にして再読込 → クラッシュせず0から起動
1. 二重加算防止（連打）
   - IDLE開始ボタンを連打しても`idleStartCount`が1回分のみ増える
   - モーダルの「続行」「休憩」を連打しても`breakCount`/`workSeconds`が二重加算されない
1. リロード安全停止
   - IDLE/WORK/BREAK中にリロード → 表示は復元されるが進行は停止、統計の二重加算なし
1. 表示破綻なし
   - 背景復帰（タブ非アクティブから戻る）でタイマー表示が急に暴走しない
